📸 How to Generate Thumbnail Images

  1. Open each template HTML file individually in Chrome (click "Open full size")
  2. Use Chrome DevTools → Ctrl+Shift+P → "Capture full size screenshot"
  3. Or use the Node.js screenshot script: node screenshot.js
  4. Convert PNG screenshots to WebP: cwebp -q 85 classic.png -o classic.webp
  5. Replace the files in /public/templates/
  6. Update templates.ts to point all templates to .webp files
